### MySql

### Laravel 4

1) Extract migration files to the migrations folder in your Laravel Project

2) Run Migrations

```
	php artisan migrate
```

3) Seed Tables

```
	php artisan db:seed
```

4) Extract model files to the models folder in your Laravel Project
